Formatter "CSS Comb".
Requires Node.js, uses CSScomb engine to sort properties in lexers CSS, SCSS, Sass, LESS. 

You can configure it in json file. Doc about config:
https://github.com/csscomb/csscomb.js/blob/master/doc/options.md

Uses portions from Sublime Text plugin:
https://github.com/csscomb/sublime-csscomb/
